# MPL
MPL的代码与示例练习

使用方法：找到对应的文件，下载即可。

- 注意**讲义左上角的标号**，比如“计算机基础–Python 基础–1”，这指示了对应文件的目录。
- 其中，**A代表例题，B代表练习**。比如“A5-1”就是“例5.1”，以此类推。
